第一条：定义

除非本协议正文另有明确所指，在本协议中所用下列术语定义如下：

1.1 亿宝贷（幸福钱庄）：指由善林金融独立运营的互联网金融信息服务平台，包括以 PC 、移动 WEB 、 APP 、微信服务号形式提供的操作环境，网址为： www.001bank.com 及其二级网页（下称“亿宝贷”）。

1.2 亿宝贷服务：指善林金融通过亿宝贷为您的借款需求而提供的相关服务。本协议中简称 “ 本服务 ” 。

1.3 指令：指您或您授权的第三方通过亿宝贷平台输入相关信息或进行点击相关动作按钮，向亿宝贷及 / 或出借人发出的各项指令，所有指令及其内容均不可撤销或变更，除非本协议另有明确约定。

1.4 额度：指亿宝贷通过其自身风控模型对您进行信用风险评估后，同意您可通过亿宝贷获得的资金上限。

1.5 募集期：您的借款需求（下称“借款标”）发布到亿宝贷页面后，亿宝贷为您的借款标设定的可供出借人选择出借的期限；

1.6 满标：募集期内，出借人所出借资金等于您所申请的借款本金。

1.7 流标：募集期内，当您的借款标无出借人选择出借或出借人所出借金小于您所申请的借款本金或因任何原因需要对借款项目取消时，亿宝贷将对您的借款标做终止募集并下线的处理。

1.8 借款期限：指您通过亿宝贷借得资金的持有期限（具体指借款的起息日至还款日）。

1.9 本金：指您通过亿宝贷审批后，出借人向您出借的未扣除相关费用的所有资金。

1.10 应还款额：指您应当偿还的借款本金、利息、费用、逾期违约金及有关应付款项（如有）的金额总和。

1.11 应付款项：指您累计未偿还的借款本金、累计已产生且未偿还的利息及其他因本服务产生或与本服务有关的款项，包括但不限于亿宝贷及 / 或出借人为您垫付的费用及实现债权的费用，如诉讼费、律师代理费、公证费、交通费、鉴定费、催收费用等。

1.12 存管银行：指在本协议为您提供充值、提现、代收、代付等资金划转服务的资金存管银行；存管账户：指您通过亿宝贷在与亿宝贷合作的存管银行开立的资金存管子账户，该账户主要用于资金保管、资金清算、账务核对等。

1.13 推荐人：指向您推荐亿宝贷，您可授权其协助您在亿宝贷上进行借款及还款操作的第三方服务机构。

第二条 借款详情

2.1 您通过亿宝贷获得的借款本金、借款期限、利率等借款详情以相关借款协议约定为准。

2.2 您在此同意，善林金融有权对您的借款申请不予通过，您将自行承担相关风险和损失。在您的借款申请获得通过并且您成功获得借款的前提下，若您的存管账户实际收到的金额与借款协议约定的借款本金不一致时，您授权善林金融委托存管银行对您的存管账户进行调账处理，以使您的存管账户收到的金额与借款本金保持一致。在收到资金超出借款本金的前提下，若您已经申请提现，但未成功，亿宝贷有权采取一切措施对您的提现指令进行拦截。若您已提现成功，则您应当在接到善林金融的通知之日将超出的金额以亿宝贷指定的方式进行操作，以返还超出的资金。否则您应当按超出金额的千分之三 / 日向善林金融支付违约金。

2.3 您知悉并同意，本次借款的起息日为您的存管账户收到借款本金之日。亿宝贷有权对您的借款标做流标处理，您的本次借款失败，本协议自始无效。

第三条：借款服务及费用

3.1 申请规则

3.1.1 您向亿宝贷提交借款申请，应当按亿宝贷确定的流程和规则，完成开户、实名认证、提交资料、填写信息以便亿宝贷对您进行风险评估。

3.1.2 您可实时在亿宝贷查询您的借款进度，若您的借款申请一旦成功提交，则您不可要求对您的借款申请进行撤回、取消。

3.1.3 开立账户：您应当依据亿宝贷的规则在亿宝贷开立平台账户，以及通过亿宝贷开立银行存管账户并绑定您本人的银行卡，以方便您接收借款资金和还款，同时您应当设置好委托扣款功能，本步骤为您通过亿宝贷进行借款的必要步骤，如果您未完成本步骤，您将无法收到借款资金并操作还款。  

3.1.4 信用风险评估：亿宝贷通过读取和分析您提交的申请信息、个人公开信息、您授权使用的个人隐私信息及其他授权信息评估您的风险承受能力，以确定您的风险等级并授予您借款额度。亿宝贷有权依据自己的独立判断决定是否通过您的借款申请，确定您的授信额度。 您知晓并同意，您的借款本金以亿宝贷风控审核通过的最终授信额度为准。

3.1.5 发布借款申请：一旦您通过了亿宝贷的风控审核，您的借款需求（下称“借款标”）即发布到善林金融运营的亿宝贷平台（网址： www.001bank.com ）供亿宝贷上的出借人用户选择出借。

3.1.6 募集期：您的借款标募集期以亿宝贷平台公示为准。若募集期内您的借款标未募集满额，则亿宝贷将自动将您的借款标终止，做流标处理。

3.1.7 放款审核：亿宝贷有权独立审核您的借款申请，代理出借人做出是否审核通过的判断。

3.1.8 借款成功：一旦您的借款标募集满额，则亿宝贷系统将自动向存管银行发出指令将所募集资金划付到您的存管账户。 资金一经支付至您的存管账户，即视为您已借款成功。

3.1.9 借款明细查阅：您可在您的亿宝贷账户中查询您借款详情，包括但不限于借款本金、利息、费用、逾期违约金等。

3.1.10 催收：您知悉并同意，善林金融受您的借款项目的债权人所托，为其提供催收服务，因催收而导致的所有费用均由您承担。且该等服务善林金融可自行提供亦可转委托第三方催收公司进行，若善林金融转委托第三方催收公司进行，则善林金融有权向该第三方催收公司披露您及相关债项的详细资料。在您无法通过亿宝贷履行还款义务的情况下，您应当依据亿宝贷的指示将应还款额支付到出借人指定的代收账户。

3.3 还款

3.3.1 您应在还款日 15:30 前清偿全部应还款（其中包括借后管理费），逾期未清偿，将影响您的征信记录。如还款日遇到法定假日或公休日，还款日期提前至法定假日或公休日的前一个工作日 15:30 前。

3.3.2 主动还款：您可于还款日及逾期后的任一时间主动登录亿宝贷平台，发起还款申请，清偿应还款项。

3.3.3 自动还款：您可于本条款第 1 项约定的时间前将应还款充值到您的存管账户或 保留在您已成功设置委托代扣功能的银行卡账户中，您在此不可撤销地授权亿宝贷委托支付机构于还款日及逾期后的任一时间（只要您的银行卡账户中有余额）将前述资金扣划至相关债权人账户以完成还款。

3.3.4 还款顺序：若您的账户资金少于应还款额时，您的还款资金将先用于偿还逾期违约金，其次支付借后管理费，再偿还利息、本金。亿宝贷有权根据您的信用状况和出借人的风险政策单方面变更前述还款顺序，相关通知发送到您的手机号即视为送达。
